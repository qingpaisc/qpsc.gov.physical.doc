## 详情
    
```
GET /admin/sms-template/:id
```
  
  
## 列表
  
```
GET /admin/sms-template
```
> 参数 检测点ID:stationId   

## 删除
  
```
Delete /admin/sms-template/:id
```
> 多个id用逗号分隔 

## 修改
  
```
Patch /admin/sms-template/:id
```

## 新增
  
```
Post /admin/sms-template
```

## 类型列表
  
```
GET /admin/sms-template/type-list
```
> 参数 检测点ID:stationId   

## 初始化自定义模板
  
```
GET /admin/sms-template/smsTemplateInit
```
> 参数 检测点ID:stationId
