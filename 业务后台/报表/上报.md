## 上报

```
POST /admin/report/up-report
```

> 请求参数: {"planId":筛查计划Id,"zone":"县（区）或者市",type:上报type}


## 查询上报任务列表

```
Get /admin/report/up-report/up
```


## 查询接收列表

```
Get /admin/report/up-report/lower
```

> 请求参数: {"planId":筛查计划ID}

## 状态修改(撤销,审核通过.退回)

```
PATCH /admin/report/up-report/change
```

> 请求参数: {"id":报告id,"status":上报状态}
> 上报状态值见全局系统参数
