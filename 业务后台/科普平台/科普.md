## 详情
    
```
GET /admin/science-doc/:id
```
  
## 列表
  
```
GET /admin/science-doc
```
> 参数 检测点ID:stationId   

## 删除
  
```
Delete /admin/science-doc/:id
```
> 多个id用逗号分隔 

## 修改
  
```
Patch /admin/science-doc/:id
```
> file:科普文件
## 新增
  
```
Post /admin/science-doc
```
> file:科普文件
